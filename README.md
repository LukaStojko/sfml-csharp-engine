# SFML C# Game Engine

A simple game engine built with SFML and C#, designed for 2D game development with essential systems for game object management, physics, and state/level management.

## Features

### Game Object Manager
- Efficient game object lifecycle management
- Update and render pipeline coordination

### Physics Manager
Comprehensive collision detection system supporting multiple shape types:
- **AABB (Axis-Aligned Bounding Box)** - Fast rectangular collision detection
- **OBB (Oriented Bounding Box)** - Rotated rectangular collision support
- **Circle Shapes** - Circular collision detection
- Collision response and resolution

### Game State/Level Manager
- State-based game flow management
- Level loading and switching
- Clean state transitions

### Additional Systems
- **Input Manager** - Centralized input handling
- **Asset Manager** - Resource loading and management
- **System Manager** - Core system coordination

## Info
- .NET 6.0 or higher (made with .NET 10.0)
- SFML.NET 3.0.0
